# Assets modernises

Ce dossier contient les ressources pretes a etre integrees au jeu.

Ne pas y placer les originaux bruts sans raison: les originaux restent dans `Resources/`.

Organisation proposee:

- `rooms/`: pieces modernisees;
- `sprites/`: personnages, objets et animations modernises;
- `sounds/`: sons nettoyes ou adaptes.

Chaque asset important doit pouvoir etre relie a sa source originale et a son workflow de creation.
